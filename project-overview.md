# Project Overview

## Project Name

Cross-border E-commerce Operations Toolkit

## Summary

This is a public portfolio of AI-assisted tools for cross-border e-commerce operations. The project focuses on turning repetitive operator workflows into structured local tools, especially spreadsheet-heavy tasks.

## Problem

Operators often need to repeatedly prepare product fields, fill marketplace templates, compare shipment quantities, and follow long SOPs. These tasks are easy to repeat incorrectly, especially when different marketplaces and warehouse systems use different field names.

## Approach

The toolkit follows an Excel-first workflow:

1. Upload source files.
2. Parse and normalize fields.
3. Apply SOP-derived rules.
4. Ask for manual confirmation where business judgment is needed.
5. Generate reviewable output files.

## Public Scope

This site only presents product design, workflow design, and privacy-safe project summaries. It does not include company files, account automation, internal SOP screenshots, or real store data.
