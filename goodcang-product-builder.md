# GoodCang Product Builder

## Purpose

The GoodCang Product Builder helps convert product source information into a warehouse bulk-upload template.

## Features

- Product file upload
- Template upload or default template usage
- Field extraction and normalization
- SKU deduplication
- Manual overrides for material, product URL, and declaration price
- Output checklist for fields requiring review

## What It Demonstrates

- Warehouse template automation
- Excel generation
- Product data cleaning
- Practical operator-focused UI design
