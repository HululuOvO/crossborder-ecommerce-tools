# Amazon Listing Helper

## Purpose

The Amazon Listing Helper supports marketplace listing preparation across multiple sites. It helps organize fields that operators need to review before creating or updating listings.

## Features

- Multi-site field defaults
- Unit conversion for dimensions and weight
- Manual confirmation for descriptions and bullet points
- Product title and model field organization
- Fulfillment option normalization
- Output designed for human review

## What It Demonstrates

- Spreadsheet parsing
- Field mapping
- SOP-to-rule translation
- Human-in-the-loop workflow design
- Validation before platform upload
