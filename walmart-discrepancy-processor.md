# Walmart WFS Discrepancy Processor

## Purpose

The Walmart WFS Discrepancy Processor helps compare monthly shipment reports with discrepancy investigation sheets.

## Features

- Extracts shipment IDs and received units
- Matches monthly shipment reports to discrepancy sheets
- Updates received quantity
- Calculates final differences
- Removes rows that no longer need investigation
- Creates a separate review sheet for removed rows

## Public Version Note

This public portfolio describes only the spreadsheet processing workflow. It excludes RPA, account login, browser automation, and internal case submission flows.
